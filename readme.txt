Тестовое задание было выполнено за 22 минуты без использования готовых решений, таких как Jquery, Boostrap, Elm, React, Vue, etc и тд!
Галерею можно использовать как на Nginx так и на Apache под управлением PHP и MySQL

Для того, чтобы портировать код, необходимо:
1) Переписать содержимое в коренную папку сервера "www"
2) Загрузить SQL дамп из файла "dump.sql" в базу данных MySQL под названием "gal" (создастся автоматически)
3) Разрешить права на запись на папку "uploads" - создать ее
4) При необходимости загрузки больших файлов, увеличить максимальные размер загружаемого фото в php.ini
5) Файл "config.php" - отвечает за подключение базы данных MySQL (его необходимо отредактировать)

По умолчанию:
Сервер MySQL: 'localhost'
Пользователь MySQL: 'root'
Пароль MySQL: ''
Название базы данных MySQL: 'gal'
